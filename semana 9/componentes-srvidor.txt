Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del sistema y permite ejecutar aplicaciones como Node.js.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad del servidor. Ayudan a detectar errores y monitorear el funcionamiento del sistema.
